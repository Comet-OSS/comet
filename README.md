# Comet

> [!WARNING]
> This README is a work in progress.

Comet is an open-source, non-intrusive command suite for users and developers alike. It is designed to be extensible and hackable so you can take control of your experience.

> [!NOTE]
> Comet does not yet have a stable release, feel free to generate your own builds, our packages are listed in [aftman.toml](./aftman.toml)!
